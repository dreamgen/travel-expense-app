## 📝 使用流程範例

### 情境：公司員工去日本旅遊 3 天

#### Day 0：出發前設定（5 分鐘）

**設定旅遊資訊：**
1. 點擊底部「⚙️ 設定」
2. 填寫：
   - 地點：日本東京
   - 日期：2025-02-01 ~ 2025-02-03
   - 補助額度：10000
3. 儲存

**新增員工（假設 4 人）：**
1. 點擊「＋ 新增員工」
2. 依序新增：
   - 王大明（申請補助：是，到職日：滿一年）
   - 李小華（申請補助：是，到職日：2024-08-01）
   - 張美美（申請補助：是，到職日：滿一年）
   - 陳小強（申請補助：否）

---

#### Day 1：第一天記帳

**08:00 機場交通**
1. 點擊首頁右下角「＋」
2. 填寫：
   - 類別：交通費
   - 日期：2025-02-01
   - 說明：桃園機場到台北車站
   - 幣別：TWD
   - 金額：150
3. 拍照上傳車票
4. 新增 ✓

**14:00 飯店住宿**
1. 點擊「＋」
2. 填寫：
   - 類別：住宿費
   - 說明：東京希爾頓飯店（第一晚）
   - 幣別：JPY
   - 匯率：0.21
   - 金額：15000
3. 拍照上傳收據
4. 新增 ✓

**19:00 晚餐**
1. 點擊「＋」
2. 填寫：
   - 類別：餐費
   - 說明：壽司店晚餐
   - 幣別：JPY
   - 金額：3500
3. 拍照
4. 新增 ✓

👉 **小技巧**：每次消費後馬上記錄，避免忘記！

---

#### Day 2-3：持續記帳

重複以上步驟，記錄所有費用：
- 🏨 住宿費用
- 🚗 交通費用（地鐵、計程車）
- 🍽️ 餐飲費用
- 🎡 景點門票（其他費用）

---

#### Day 4：回國後整理（2 分鐘）

1. 檢查所有費用是否都已記錄
2. 確認單據照片都有上傳
3. 點擊「⚙️ 設定」→「📄 產生 Excel 申請單」
4. 下載檔案
5. 完成！🎉

## 💡 實用技巧

### 📸 拍照技巧
- 光線充足的地方拍攝
- 收據攤平，避免皺褶
- 確保金額、日期清楚可見
- 可以拍多張角度後選最清楚的

### 💰 幣別換算
- 即時匯率可用手機查詢
- 建議使用當天匯率
- 或統一用一個固定匯率

### 📊 分類建議
- **代收轉付**：團體代收代付的費用
- **住宿費**：飯店、民宿
- **交通費**：機票、高鐵、計程車、租車
- **餐費**：早中晚餐、下午茶
- **其他費用**：門票、購物、雜費

### 🔄 多人協作
- 方案一：各自記錄，最後匯總
- 方案二：一人負責，其他人提供單據
- 方案三：分類負責（住宿、交通、餐飲）

## 🎯 常見場景

### 場景 1：用外幣付款
**問題**：在日本用日幣付款，怎麼記錄？

**解答**：
1. 選擇幣別「JPY」
2. 輸入日幣金額（如 3000）
3. 匯率會自動設定（0.21）
4. 台幣金額自動計算（3000 × 0.21 = 630）

### 場景 2：忘記拍單據
**問題**：費用已經記錄了，但忘記拍照怎麼辦？

**解答**：
1. 可以之後補拍
2. 或回國後掃描紙本單據
3. 也可以不上傳照片，只記錄數字

### 場景 3：團體代付
**問題**：一個人先付了所有住宿費，怎麼記錄？

**解答**：
1. 類別選「代收轉付收據」
2. 說明寫：「王大明代付全體住宿費」
3. 金額填總金額
4. 在 Excel 中會自動計算分攤

### 場景 4：更改費用
**問題**：輸入錯誤，想修改怎麼辦？

**解答**：
1. 目前只能刪除重新輸入
2. 點擊費用卡片右上角垃圾桶
3. 確認刪除
4. 重新新增正確資料

## 🆘 遇到問題？

### 無法拍照
➡️ 確認使用 HTTPS 網址（以 https:// 開頭）

### 資料不見了
➡️ 檢查是否清除了瀏覽器資料或換了瀏覽器

### Excel 下載失敗
➡️ 檢查瀏覽器是否阻擋下載

### 匯率不對
➡️ 可以手動修改匯率欄位
