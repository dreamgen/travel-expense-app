# 🚀 部署指南

將旅遊費用申請 APP 部署到網路上，讓員工可以隨時隨地使用！

## 方案一：GitHub Pages（推薦）

### 步驟：

1. **建立 GitHub 帳號**
   - 前往 https://github.com
   - 註冊或登入

2. **建立新專案**
   - 點擊右上角 `+` → `New repository`
   - 名稱：`travel-expense-app`
   - 設定為 Public
   - 點擊 `Create repository`

3. **上傳檔案**
   - 點擊 `uploading an existing file`
   - 拖曳 `travel-expense-app.html` 和 `app.js`
   - Commit changes

4. **啟用 GitHub Pages**
   - 進入 `Settings` → `Pages`
   - Source 選擇 `main` 分支
   - 點擊 `Save`
   - 等待 1-2 分鐘

5. **取得網址**
   - 網址格式：`https://你的帳號.github.io/travel-expense-app/travel-expense-app.html`
   - 分享給同事使用！

### 優點：
- ✅ 完全免費
- ✅ 可自訂網址
- ✅ 穩定可靠
- ✅ 支援 HTTPS

---

## 方案二：Netlify（最簡單）

### 步驟：

1. **前往 Netlify**
   - 網址：https://www.netlify.com

2. **拖曳部署**
   - 登入後點擊 `Add new site` → `Deploy manually`
   - 建立一個資料夾，放入兩個檔案
   - 直接拖曳資料夾到網頁上
   - 自動部署完成！

3. **取得網址**
   - 會自動產生隨機網址
   - 可以自訂子網域名稱

### 優點：
- ✅ 最快速（1 分鐘完成）
- ✅ 拖放即可
- ✅ 自動 HTTPS
- ✅ 可自訂網域

---

## 方案三：Vercel

### 步驟：

1. **前往 Vercel**
   - 網址：https://vercel.com

2. **連接 GitHub**
   - 用 GitHub 帳號登入
   - Import 你的專案
   - 自動部署

### 優點：
- ✅ 與 GitHub 整合
- ✅ 自動更新
- ✅ 快速部署

---

## 方案四：本地測試（開發用）

### 使用 Python 啟動本地伺服器：

```bash
# Python 3
cd 專案資料夾
python -m http.server 8000
```

然後開啟：http://localhost:8000/travel-expense-app.html

### 使用 Node.js：

```bash
# 安裝 http-server
npm install -g http-server

# 啟動
http-server
```

## 🔒 安全性建議

1. **HTTPS 必須**
   - 使用拍照功能需要 HTTPS
   - 上述方案都提供免費 HTTPS

2. **資料隱私**
   - 所有資料儲存在使用者本地
   - 不會上傳到伺服器
   - 適合處理公司內部資料

3. **訪問控制**（選配）
   - 可在檔案前加入簡單密碼驗證
   - 或使用公司內部網路部署

## 📱 設定為 PWA（Progressive Web App）

如果要讓 APP 可以「安裝」到手機：

1. **建立 manifest.json**

```json
{
  "name": "員工旅遊費用申請",
  "short_name": "旅遊費用",
  "description": "快速記錄旅遊費用並產生申請單",
  "start_url": "/travel-expense-app.html",
  "display": "standalone",
  "background_color": "#ffffff",
  "theme_color": "#667eea",
  "icons": [
    {
      "src": "icon-192.png",
      "sizes": "192x192",
      "type": "image/png"
    },
    {
      "src": "icon-512.png",
      "sizes": "512x512",
      "type": "image/png"
    }
  ]
}
```

2. **在 HTML 中加入**

在 `<head>` 區塊加入：

```html
<link rel="manifest" href="manifest.json">
<meta name="theme-color" content="#667eea">
```

3. **建立 Service Worker**（選配，可離線使用）

## 🎯 推薦配置

→ 使用 Netlify 拖放部署

**中型公司（50-200 人）**
→ 使用 GitHub Pages + 自訂網域

**大型企業（> 200 人）**

## ⚙️ 自訂網域設定

### GitHub Pages：
1. 在 `Settings` → `Pages` 中設定 Custom domain
2. 在域名商設定 CNAME 記錄

### Netlify：
1. `Domain settings` → `Add custom domain`
2. 依照指示設定 DNS

## 🆘 疑難排解

### 相機功能無法使用
- 確認使用 HTTPS
- 檢查瀏覽器權限設定

### 無法開啟檔案
- 確認兩個檔案在同一資料夾
- 使用伺服器而非直接開啟檔案

### Excel 下載失敗
- 檢查瀏覽器是否阻擋下載
- 確認已允許彈出視窗
